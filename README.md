# 🎰 Slot Machine Game (Python Project)

## 📌 Overview

This is a simple command-line Slot Machine game built in Python. The project simulates a real casino-style slot machine where users can deposit money, place bets, and spin to try their luck. The game uses random number generation to determine outcomes and applies basic betting logic to calculate wins and losses.



---

## 🎮 How the Game Works

- The player starts by depositing money  
- The player chooses:  
  - Number of lines to bet on  
  - Amount to bet per line  
- The slot machine generates a random 3x3 grid of symbols  
- If symbols match on a line:  
  - The player wins based on the symbol value  
- Otherwise:  
  - The bet amount is lost  
- The game continues until the player:  
  - Runs out of balance OR  
  - Chooses to quit  

---

## 🧠 Features

- 💰 Deposit system to start the game  
- 🎯 Flexible betting system (lines + bet amount)  
- 🎰 Random slot machine spin generator  
- 🧮 Win/loss calculation logic  
- 🔁 Continuous gameplay loop until exit
